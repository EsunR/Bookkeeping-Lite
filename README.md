<center><h1>说明</h1></center>

本项目为 “记账薄·轻” 的前端项目源码以及设计文稿，仅供参考学习使用，附带后台 API 接口文档，可按照规范自行实现，API 端口请在 `/src/common.vue` 下修改。

安装依赖：

```
npm install
```

运行开发环境：

```
npm run serve
```

编译：

```
npm run build
```
